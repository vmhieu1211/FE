        for( biểu thứ 1; biểu thức 2; biểu thứ 3) {
            code trong vòng lặp 
        }  
        Expression 1 is executed (one time) before the execution of the code block.
        Biểu thức 1 thực thi 1 lần duy nhất trước khi code trong {} của vòng lặp được chạy
        
        Expression 2 defines the condition for executing the code block.
        Biểu thứ 2 định nghĩa điều kiện để chạy vòng lặp 
        
        Expression 3 is executed (every time) after the code block has been executed.
        Biểu thức 3 được chạy ( mỗi lần lặp) sau khi code trong vòng lặp được chạy
